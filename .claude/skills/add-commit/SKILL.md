# Add and Commit

Stage all current changes and create a commit with a descriptive message.

1. Run `git status` and `git diff --cached` and `git diff` to understand what changed.
2. Stage the relevant files. Prefer adding specific files by name over `git add -A`.
3. Write a concise commit message that describes what changed and why.
4. Commit.
